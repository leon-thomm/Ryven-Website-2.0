Who is Ryven for?

About this tool

What is Ryven

Ryven enables using the more diagram-like dataflow inspired representation to visualize your scripts. It runs code. You should know the Python programming language and its basic concepts. Ryven provides a simple system to create nodes that can execute any Python code, and add intuitive GUI. Use cases range almost through the whole Python domain, from IoT, over any kind of data processing, Raspberry Pi, basically any web API, generative engineering, and lots of stuff that hasn't even been tried yet.

What it's not

A tool to learn how to program or to generally program in Python. It does not automatically generate graphs out of existing code. It does not try to conceptually replace source code by the visual graphs. Both representations have their domains and, even though it enables significant abstraction, Ryven tries to make them work together in an efficient way. Please also keep in mind that Ryven is not a professional piece of software and that it's still receiving quite active development and sometimes major changes.

Create clear and pretty flows

Design, of course, plays a huge role. Therefore, Ryven offers a variety of different flow designs.
